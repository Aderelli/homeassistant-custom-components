### edp_redy:
Copy the following files to your custom_components folder (creating the requires subfolders): 
- edp_redy.py
- sensor/edp_redy.py
- switch/edp_redy.py

*Do not add sensor/edp_redy_**local**.py since it is not used by this component.*

Add the following configuration:

```
edp_redy:
  username: 'xxxxx'
  password: 'xxxxx'
```
